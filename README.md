
# 🌾 Smart AI Agriculture Assistant

## 📌 Project Overview
This project recommends the most suitable crop using Machine Learning based on soil nutrients and weather conditions.

## 🚀 Features
- Crop Recommendation
- Confidence Score
- Fertilizer Recommendation
- Irrigation Schedule
- Temperature Range
- Rainfall Range
- Soil pH Information
- Water Requirement
- Farmer Tips

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 🤖 Machine Learning Algorithm
Random Forest Classifier

## 📊 Model Accuracy
99.30%

## 📥 Input Parameters
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

## 📤 Output
- Recommended Crop
- Confidence Score
- Fertilizer Recommendation
- Irrigation Schedule
- Farmer Tips
